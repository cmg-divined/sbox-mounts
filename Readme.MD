# s&box Mounts

A community-maintained list of known s&box mounts and related projects, along with what each one currently supports.

## Usage Note

For a mount to work in sandbox mode, the mount must be compiled and dropped into the `mount/` folder, along with any shaders that come with it in their corresponding folders.

## Mount Notes

- Garry's Mod supports all currently planned asset types except sounds and animations.
- Half-Life 2 credits to Asphaltian. This mount is actually located in the native s&box folder since this is a fork.
- Skyrim supports all currently planned asset types except sounds and animations.
